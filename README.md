# niz-cli

> Filter out available npm names through some restrictions.

**IN WINDOWS, DON'T USE CMD!**

## Install

```shell
$ npm i -g niz-cli
```

## Usage

```shell
$ niz <length> [options]
```

## Options

```shell
  --regex, -r     filter out npm names through regex
```

## Example

```shell
$ niz 3 --regex ^r
```
